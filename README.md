# Pomodoro Timer

> A simple, configurable pomodoro timer 

## Stack

- [Vue 3](https://v3.vuejs.org/)
- [Pinia](https://pinia.esm.dev)
- [Vite](vitejs.dev/)
- [Tailwind CSS](https://tailwindcss.com/)
- [VueTypes](https://dwightjack.github.io/vue-types/)

## Development

Clone the repo then:

```
npm ci
npm run dev
```
